MenuButton

A component for the common dropdown menu button design pattern.

It takes exactly two children: button itself that serves as a trigger and ListBox component with Options inside.

Basic usage

import { MenuButton, Button, ListBox, Option, Icon } from '@fxtrot/edge';

Placement

You can control the placement of the dropdown, using popper placement values.

# Icon

Uses [Material Design Icons](https://material.io/resources/icons/?style=baseline) (Apache License 2.0)

## Basic usage

```tsx
import { Icon } from '@fxtrot/edge';
```

<Playground>
  <Icon icon="build" />
</Playground>

# ListBox

Popup for dropdown components like [`MenuButton`](/fx-menubutton) and [`Select`](/fx-select).
[`Select`](/fx-select) wraps it's children in `ListBox` automatically, so it should only be used with `MenuButton` component.

## Basic usage

```tsx
import { MenuButton, Button, ListBox, Option, Icon } from '@fxtrot/edge';
```

See [`MenuButton`](/fx-menubutton#basic-usage) example.

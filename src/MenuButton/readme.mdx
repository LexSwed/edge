# MenuButton

A component for the common dropdown menu button design pattern.

It takes **exactly** two children: button itself that serves as a trigger and [`ListBox`](/fx-listbox) component with [`Option`](/fx-option)s inside.

## Basic usage

```tsx
import { MenuButton, Button, ListBox, Option, Icon } from '@foxtrot/edge';
```

<Playground>
  <MenuButton>
    <Button size="s">
      <Icon icon="more_horiz" />
    </Button>
    <ListBox>
      <Option onSelect={() => alert('Edit')}>
        <Icon icon="edit" tone="outlined" />
        Edit
      </Option>
      <Option onSelect={() => alert('Remove')}>
        <Icon icon="delete" tone="outlined" /> Remove
      </Option>
    </ListBox>
  </MenuButton>
</Playground>

## Placement

You can control the placement of the dropdown, using [popper](https://popper.js.org/docs/v2/constructors/#options) `placement` values.

<Playground>
  <MenuButton placement="bottom-end">
    <Button tone="accent">Show menu</Button>
    <ListBox>
      <Option>Perform action 1</Option>
      <Option>Option 2</Option>
    </ListBox>
  </MenuButton>
</Playground>

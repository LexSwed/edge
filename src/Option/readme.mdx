# Option

Option component is used inside dropdowns.

- `value` property should be specified if used within [`Select`](/fx-select) component
- `onSelect` callback should be used to handle `Option` click within for example [`MenuButton`](/fx-menubutton)

## Basic usage

```tsx
import { MenuButton, Button, ListBox, Option } from '@fxtrot/edge';
```

<Playground>
  <MenuButton>
    <Button size="s">
      <Icon icon="more_horiz" />
    </Button>
    <ListBox>
      <Option onSelect={() => alert('Edit')}>
        <Icon icon="edit" tone="outlined" />
        Edit
      </Option>
      <Option onSelect={() => alert('Remove')}>
        <Icon icon="delete" tone="outlined" /> Remove
      </Option>
    </ListBox>
  </MenuButton>
</Playground>

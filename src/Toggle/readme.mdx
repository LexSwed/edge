# Toggle

## Basic usage

```tsx
import { Toggle } from '@fxtrot/edge';
```

<Playground>
  {() => {
    const [toggle, setToggle] = React.useState(false);
    return <Toggle checked={toggle} label="Default toggle" onChange={() => setToggle(!toggle)} />;
  }}
</Playground>
